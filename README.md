Gestor de Recetas – Python

Aplicación de escritorio desarrollada en Python para la gestión integral de recetas, productos y materias primas, orientada a pequeños emprendimientos gastronómicos o fines académicos.
El sistema permite administrar costos, recetas compuestas y productos finales mediante una interfaz gráfica y una base de datos local.

Funcionalidades principales
- Gestión de productos finales:
  - Alta, modificación, consulta y eliminación
  - Cálculo automático de costos
- Gestión de materias primas:
  - Materias primas compradas y elaboradas
  - Actualización de precios y cantidades
  - Control de dependencias (una materia prima puede ser insumo de otra)
- Gestión de recetas:
  - Asociación de productos con sus materias primas
  - Cálculo automático de costos según cantidades
  - Prevención de eliminaciones inválidas
- Interfaz gráfica intuitiva:
  - Navegación por menús
  - Mensajes de validación y control de errores
---

Tecnologías utilizadas:
- Python 3
- Tkinter (interfaz gráfica)
- SQLite (base de datos)
- Programación estructurada
- Manejo de excepciones y validaciones
---

Cómo ejecutar el proyecto?
1. Clonar el repositorio:
  git clone https://github.com/alemanyemi/recipe-manager-python
2. Acceder al directorio del proyecto:
  cd recipe-manager-python
3. Ejecutar el archivo principal:
  python "Gestor de Recetas.py"
